# Japam - Audio Files for Mantra Playback

Place these audio files in `public/sounds/`.

**Background music:** `background.mp3` (loops during gameplay) Each file plays when that deity's candies are matched.

| # | Candy (Deity) | Mantra to Record | Filename |
|---|---------------|------------------|----------|
| 1 | Rama | Ram | `rama.mp3` |
| 2 | Shiva | Om Namah Shivaya | `shiva.mp3` |
| 3 | Ganesh | Om Gan Ganapataye Namah | `ganesh.mp3` |
| 4 | Surya | Om sooryaya Namaha | `surya.mp3` |
| 5 | Shakthi | Sri Maatre namaha | `shakthi.mp3` |
| 6 | Krishna | Om namo Bhagavathe vaasudevaya | `krishna.mp3` |
| 7 | Shanmukha | Om Saravana Bhavaya Namaha | `shanmukha.mp3` |
| 8 | Venkateswara | Om namo Venkateshaaya | `venkateswara.mp3` |
