# Japam – Translation Verification Guide

## INSTRUCTIONS FOR THE VERIFIER

**Purpose:** Verify that all translations convey the correct meaning in each language. Use **English** and **Telugu** as reference languages. You must be fluent in the language you are verifying.

**What to check:**
1. **Accuracy:** Does the translation mean the same as the English/Telugu reference?
2. **Register:** Is the tone respectful and formal (appropriate for a spiritual app)?
3. **Natural phrasing:** Does it read naturally to a native speaker?
4. **No transliterations:** Avoid using English words where a native equivalent exists.
5. **Deity names:** Verify they are the standard, respectful forms used in that language/region.

**How to verify:**
- For each key, compare the translation in your language with the **English** and **Telugu** columns.
- Mark ✅ if correct, ❌ if wrong, and ⚠️ if questionable (needs review).
- Add suggested corrections in the "Notes" column if needed.

**Important:** This app is used for mantra/japa practice. Terminology around “japa,” “marathon,” “moves,” etc. should be consistent and culturally appropriate.
